## 0.1.0

- Initial release
- GPU-accelerated sprite rendering using fragment shaders
- Spatial binning for efficient batch rendering
- Texture atlas building with MaxRects packing algorithm
- Sprite trimming and whitespace optimization
- Support for sprite flipping (flipX, flipY)
- Rotation support for packed sprites
- ZIP-based atlas serialization and loading
- Isolate-based atlas building for non-blocking operations
